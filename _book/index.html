第一步先找到自己想翻译的部分

这里我们使用著名的 Trello 避免重复劳动和各个进度的管理

点击进入这个地址 Gradle Scrum Board

根据里面的说明选取想翻译的部译或校对的任务

第二步把代码叉到自己的仓库里

  1. fork主仓库到自己的仓库里
  2. 然后 git clone 到自己的电脑上
  3. 推荐大家使用 Gitbook 专属的编辑器 地址 .请注意, 每一页的创建都必须是用英文, 再通过 rename 改成中文.
  4. 如果是想学习Markdown 纯手工的同学可以参考这里学习 地址1 地址2
  5. 修改完 git commit 到自己的仓库
  6. 给主仓库的 master 分支提交 pull request
  7. 然后等待其他人的审核 8 审核修改完成之后你翻译的部分就会被合并到书里
